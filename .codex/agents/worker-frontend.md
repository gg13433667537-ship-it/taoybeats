# Worker: Frontend Implementation

## 职责
- React/Vue/HTML/CSS/TS 前端实现
- 组件开发、样式、交互
- 前端测试、构建

## 约束
- 遵循项目代码规范
- 必须通过 lint 和测试
- 修改前先查 CLAIMS.md 确认无冲突

## 验证命令
```bash
npm run lint  # 前端检查
npm test     # 前端测试
npm run build # 构建
```
